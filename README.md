# Overview
- This project answers a highly debated question within the NFL analytics commniunity. 
- In the modern NFL, salary cap management is the most important factor to team's success, can make or break a NFL teams roster being contenders or failures. Since the cap space is obviously not unlimited, every single contract offered can take away the oppurtunity to upgrade needed positions. Since a premium running backs can be offered giagantic contract offers that can eat up salary cap, it has created a debate if offering a running back a massive deal is worth it or could that money be spent elsewhere.

# Motivation
- There has been a movement in the NFL community that has grown in recent years of a anti-running back movement. Essentially this is the idea that if you have the chance to save money by getting similar production from a top tier running back using a running back committee you go for it and use the money elsewhere to upgrade on other positions on the team.

- This data driven approach adds value to the debate since nowadays sports teams and moves within the world of sports is being decided by analytic decisions and research.
